 6:  * Holds data for a point.
 7:  *
 8:  * This class has a precision property that will
 9:  * round all point data to this precision.
10:  * By default it's 3, rounding everything
11:  * to 1/1000th of a mm, which is very accurate.
12:  * If you increase the precision, some operations
13:  * (like path offset) will become rather expensive (slow)
14:  * because mathematically, they are approximations.
15:  * And making the approximation overly precise is just being
16:  * difficult for the sake of being difficult.
17:  * For all intends and purposes, 1/1000th of a mm is way more
18:  * precice than you'll even be able to sew or cut.
19:  *
20:  * @author Joost De Cock <joost@decock.org>
21:  * @copyright 2016 Joost De Cock
22:  * @license http://opensource.org/licenses/GPL-3.0 GNU General Public License, Version 3
23:  */
