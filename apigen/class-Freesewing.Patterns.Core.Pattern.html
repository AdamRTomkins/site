# pileParts( )

Stacks all parts in the top left corner in preparation of layout

Stacks all parts in the top left corner in preparation of layout

When you draw a part, there's no reason it would have point (0,0) as its top left corner. This takes care of that by pushing everything to the top left (by adding a translate transform). We do this before we layout the pattern with our packer.

# layoutTransforms( array $layoutBlocks )

Add transforms to parts to implement layout calculated by the packer

Add transforms to parts to implement layout calculated by the packer

The packer does not actually layout the pattern. It merely calculates it and returns an array of 'layoutblocks' that contain the needed information on how to layout the different parts. This takes that info and applies a tranform to the parts for it, thereby making implementing the calculated layout in the SVG document.

$layoutBlocks
array
$layoutBlocks Array of layoutblocks calculated by the packer
