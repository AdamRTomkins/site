# handleInvalidRequest( Freesewing\Context $context )

What to do when a request is considered to be invalid

What to do when a request is considered to be invalid

If you return false in isValidRequest() then we need to do something with the ongoing request. Since you decided it's no good, you get to decide what to do with it.
