# initialize( Freesewing\Model $model )

Sets up options and values for our draft

Sets up options and values for our draft

By branching this out of the sample/draft methods, we can set a bunch of options and values the influence the draft without having to touch the sample/draft methods When extending this pattern so we can just implement the initialize() method and re-use the other methods.

Good to know: Options are typically provided by the user, but sometimes they are fixed Values are calculated for re-use later

# draftExamplePart( Freesewing\Model $model )

Drafts the examplePart

Drafts the examplePart

We are using a draft[part name] scheme here but don't let that think that this is something specific to the draft service.

This draft method does the basic drafting and is called by both the draft AND sample methods.

The difference starts after this method is done. For sample, this is all we need, but draft calls the finalize[part name] method after this.
