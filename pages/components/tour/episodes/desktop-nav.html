Going home

On every page, you'll find our slanted page header.

Click the Freesewing header or our logo to return to the home page.

Menu

At the lower end of the page header sits the menu.

The menu is the main way to nagivate the site.

Breadcrumbs

Breadcrumbs near the top of the page show you where you are.

You can tap these breadcrumbs to navigate. The first breadcrumb always takes you to the home page.

Modal windows

Some content, like the login form, loads in a popup that covers the screen.

Click the cross icon in the bottom right corner to close them, and return to the page underneath.

This is the end of this guide

Remember, the menu is at the top, modal windows close at the bottom right.
